What is Pyastrator?

Pyastrator is a simple web-based tool that allows you to draw and animate vector graphics. It is built using Python and the Flask web framework.

How to use Pyastrator?

Simply click on the canvas to draw a shape. You can also use the toolbar on the left to select a different shape or color. To animate the shape, click on the "Animate" button.

Features

  • Draw and animate vector graphics
  • Select different shapes and colors
  • Export your drawing as an SVG file
